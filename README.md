GitBook Table Of Content Plugin
==============

为GitBook生成目录结构，效果如下：

![gtoc效果图](https://lh6.googleusercontent.com/-jsqrsB5Pu1o/VJLHr7s0J2I/AAAAAAAACNw/f3jcM5F4aVc/s800/gtoc%25E6%2588%25AA%25E5%259B%25BE.png)



### 使用方式
在`book.json`中添加如下配置：
{
    "plugins": ["gtoc"]
}

然后运行`gitbook install`即可。

### 功能
 - 抽取页面标题组成目录列表
 - 自动添加序号
 - 可隐藏

### 说明
 - 默认只抽取h2,h3标题
 - 默认进入页面就显示（后期会修改默认隐藏）
